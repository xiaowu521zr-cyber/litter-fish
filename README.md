# 仿生机器鱼控制系统 (Biomimetic Robotic Fish Control System)

基于 STM32F407 + FreeRTOS 的水下仿生机器鱼嵌入式控制系统，实现多传感器数据融合、多模通信、舵机/推进器协同控制以及 PID 巡航等核心功能。

---

## 📋 项目概述

本项目是一个水下仿生机器鱼的实时嵌入式控制系统。系统以 **STM32F407VET6**（ARM Cortex-M4, 168MHz）为主控芯片，搭载 **FreeRTOS** 实时操作系统，通过多任务并行架构实现对机器鱼的姿态感知、运动控制、深度调节及远程通信等功能。

## 🧵 FreeRTOS 任务架构

| 任务名称 | 优先级 | 栈大小 | 功能描述 |
|---------|--------|--------|---------|
| **defaultTask** | Normal | 512B | 🎯 主控制循环：舵机游动控制 + 深度PWM调节 + 巡航模式切换 |
| **Sensor_Task** | Normal | 8KB | 📊 传感器采集：循环读取 IMU、深度、电源、视觉数据并打印 |
| **Parse_Task** | Normal1 | 1.5KB | 🔍 指令解析：解析并分发控制指令到对应模块 |
| **Status_Task** | Low2 | 1KB | 📡 状态监控：监控系统运行状态 |

### 任务间通信

```
串口中断 ──► uart3_rx_queue ──► CommReceiveEntry (Receive_Task)
                                     │
                                     ▼ parse_command()
                              ┌──────┴──────┐
                              │  D 系列指令  │──► Flag_Servo / turn_direction (全局标志)
                              │  X 系列指令  │──► ControlPropellers()	电机调节实现小鱼上浮下沉
                              │  Z 系列指令  │──► Flag_Z (重心滑块调节)
                              │  C 系列指令  │──► Cruise_Start/Stop()
                              │  P 系列指令  │──► PID 参数调节
                              │  T 系列指令  │──► 转弯速率调节
                              │ GA/S4 指令   │──► servo_cmd_queue
                              └─────────────┘
```

## 🎮 控制指令集

### D 系列 — 运动方向控制

| 指令 | 功能 | 说明 |
|------|------|------|
| `D0` | 🛑 停止 | 舵机停止，方向复位 |
| `D1` | ⬆️ 前进 | 仿生游动前进模式 |
| `D2` | ⬅️ 左转 | 转向游动左转模式 |
| `D3` | ➡️ 右转 | 转向游动右转模式 |

> 发送 D 系列指令会自动退出巡航模式。

### X 系列 — 推进器控制

| 指令 | 功能 |
|------|------|
| x0   | 电机停止             |
| `X1` | 电机正转（小鱼下沉） |
| `X2` | 电机反转（小鱼上浮） |

### Z系列 — 深度档位控制 (PWM 占空比)

| 指令 | PWM 脉宽 | 说明 |
|------|---------|------|
| `Z0` | 4.5% | 滑块从初始状态向后滑 |
| `Z1` | 5.5% | 比Z0向后滑的大一些 |
| `Z2` | 6.5% | 比Z1向后滑的大一些 |
| `Z3` | 7.5% | 默认初始档位 |
| `Z4` | 8.5% | 滑块向前 |
| `Z5` | 9.5% | 比Z4大一些 |
| `Z6` | 10.5% | 都是滑块向前 |
| `Z7` | 11.5% | |
| `Z8` | 13.5% | |
| `Z9` | 35.5% | 滑块处于最前方 |

### C 系列 — 巡航控制

| 指令 | 功能 |
|------|------|
| `CL` | 启动左转巡航模式 |
| `CR` | 启动右转巡航模式 |
| `CS` | 停止巡航 |

### P 系列 — PID 参数切换

| 指令 | Kp | Ki | Kd | 说明 |
|------|-----|-----|-----|------|
| `P0` | 0.5 | 0.01 | 0.05 | 柔和参数组 |
| `P1` | 0.8 | 0.02 | 0.10 | 适中参数组 |
| `P2` | 1.2 | 0.03 | 0.15 | 激进参数组 |

### T 系列 — 目标转弯速率

| 指令 | 速率 |
|------|------|
| `T0` ~ `T5` | 30°/s ~ 55°/s（每档 +5°/s） |

## 🔧 核心功能模块

### 1. 传感器数据融合

`Sensor_Task` 以 **100Hz** 频率循环采集所有传感器数据并通过串口输出：

```json
{
  "fish_attitude": {
    "roll": ..., "pitch": ..., "yaw": ...,
    "roll_speed": ..., "pitch_speed": ..., "yaw_speed": ...
  },
  "depth_sensor": {
    "depth": "...cm", "temperature": "...°C", "pressure": "...Pa"
  },
  "power_sensor": {
    "voltage": "...mV", "current": "...mA", "power": "...mW"
  }
}
```

### 2. PID 巡航控制

基于 IMU 偏航角速度反馈的 PID 闭环巡航系统：
- 支持左转/右转巡航模式
- 预设 3 组 PID 参数（柔和/适中/激进）
- 目标转弯速率可在 30~55°/s 范围内调节
- 巡航模式下自动覆盖手动方向控制

### 3. 舵机仿生游动

通过多路 PWM 信号驱动仿生鱼鳍/尾鳍，实现：
- 正弦波仿生游动（前进模式）
- 非对称摆幅转向（左转/右转模式）
- 状态机时序控制（Duty1 → Duty2 → Duty3 → 循环）

### 4. 线程安全机制

- `uartMutex` — 串口输出互斥锁（防止多任务 printf 竞争）
- `servoMutex` — 舵机控制互斥锁（保护方向标志位）
- `cruiseMutex` — 巡航状态互斥锁（保护巡航控制结构体）
- 消息队列 — 串口接收数据通过 FreeRTOS 队列异步传递

## 🔄 并发控制流程

```
defaultTask (主控制循环, ~100Hz):
    1. UpdatePropellers()          # 更新推进器状态
    2. 获取 servoMutex             # 读取方向标志
    3. 获取 cruiseMutex            # 读取巡航状态
    4. 判断模式:
       ├─ 巡航激活 → Cruise_Update()
       └─ 手动模式 → Servo_ForwardTask() / Servo_UpdateTask() / Servo_Stop()
    5. 深度控制 (Flag_Z)
    6. osDelay(10ms)
```

## ⚙️ FreeRTOS 配置

- **内核频率**: 1000Hz (1ms tick)
- **最小栈大小**: 128 words
- **总堆大小**: 75KB (configTOTAL_HEAP_SIZE)
- **最大优先级**: 56
- **任务调度**: 抢占式 (Preemptive)
